# 電車遅延お知らせ BOT

---

## 概要

兵庫～大阪間の電車の遅延情報をお知らせする LINE Bot です。

**※このサービスは動作しません。**

- 平日の通勤の時間帯（7 ～ 10 時と 17 ～ 19 時の間）に遅延が発生していれば通知します。
- 送信されたメッセージ内容に応じて遅延情報の返答もできます。
- [鉄道遅延情報の json](https://rti-giken.jp/fhc/api/train_tetsudo/) さんにアクセスし、取得結果をもとにメッセージを作成しています。
